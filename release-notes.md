## ZimbaBeats Family v1.0.35

Parental Control Companion App for ZimbaBeats

### What's New
- **CI/CD Improvements**: Enhanced release automation using GitHub CLI
- **Security**: Added security policy documentation

### Downloads
- **ZimbaBeats-Family.apk** - Parental Control Companion App

### Installation
1. Download ZimbaBeats-Family.apk
2. Enable "Install from unknown sources" in Android Settings
3. Install ZimbaBeats-Family.apk on parent's device
4. Download and install ZimbaBeats.apk from the separate v1.0.35 release
5. Pair the apps and configure parental controls

### Requirements
- Android 8.0 (API 24) or higher
- Main app targets Android 14 (API 36)
- Family app targets Android 14 (API 34)

### Features
- Safe YouTube music streaming for kids
- Bedtime scheduling (per-child)
- Content filtering (blocked keywords/channels per-child)
- Multi-child profile management
- Real-time sync between apps
- In-app update mechanism
